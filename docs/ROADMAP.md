# Roadmap

PolySIEM is focused on practical documentation and security context for self-hosted environments. Planned areas include:

- Network scanner discovery for unmanaged LAN hosts.
- Richer IP address management views, including subnet utilization and conflict detection.
- JSON and CSV import/export for inventory and documentation.

Priorities may change as the project develops. Track current work and propose features in the [GitHub issue tracker](https://github.com/Realynx/PolySIEM/issues).
