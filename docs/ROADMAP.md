# Roadmap

PolySIEM stays focused on practical documentation and security context for self-hosted environments. Here's what's on the list:

- Network scanner discovery for unmanaged LAN hosts.
- Richer IP address management views, including subnet utilization and conflict detection.
- JSON and CSV import/export for inventory and documentation.

Priorities shift as the project develops. The [GitHub issue tracker](https://github.com/Realynx/PolySIEM/issues) is the place to see what's actually in flight or to propose a feature.
